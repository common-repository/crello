== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use "Crello settings" link to enter your api key you got on crello.com
4. Ensure that uploads directory is writable

== Description ==

Create With Crello Plugin
=========================
Integrate a free graphics editor with 12,000+ templates into your WordPress blog. 
=================================================================================

Create in Crello is a free WordPress plugin that will take care of all your visual content needs. It provides instant access to the graphics editor, where you can easily create visuals for your blog post. 

Crello has 12,000+ free graphics templates, designed by professionals and available in the plugin, as well as thousands of high-res photos, icons, illustrations, frames, stickers etc. 

== How does the Crello plugin work? ==

&#49;. Install the plugin
=========================

You can find full installation instructions [here](https://wordpress.org/plugins/crello/#installation).
Don’t forget to choose the format you would like to use (although you can always change it later).

There are 30 graphics formats that you can choose from, including:

Facebook, Instagram, Twitter, and Pinterest posts
Social Media covers
Blog Titles and Graphics
Email headers
Banner ads
Postcards, Posters and Cards

&#50;. Open Crello editor
=========================

Click on the button &#91;Create in Crello&#93; at the top of the post editing window. 

The plugin will open the Crello editor with your artboard and all the essential design tools.

&#51;. Choose a template.
=========================

There are thousand of templates you can select from. Use Search by Category to quickly find thematic designs. 

&#52;. Customize your design
============================

You can upload your photos, change the text and all the design elements, add a logo, and create designs from scratch. Whatever works for you!

A couple of tips: 

* In the “Objects” tab, there are thousand of stickers, frames, borders and icons that will make your design look even more professional.

* To find free high-res photos, go to “Photos” - “Free Photos” tab.

* Use Text Badges from the “Text” tab to embed words into images for a more trendy look. 

&#53;. Publish the image
========================

Just click on the &#91;Publish&#93; button in the upper right hand corner to add the image to your post. 


If you have questions or problems with the Crello plugin, [contact the Crello team](https://crello.com/faq/suggest_a_feature/). 

== Instructions ==

== How can I install the Crello plugin? ==

Option 1
--------
Login to your WP blog and go to the Plugins menu 
Click the “Add New” button
Search “Crello”
Click the &#91;Install&#93; button
Click the &#91;Activate&#93; button

Option 2
--------
Go to [https://wordpress.org/plugins/crello/](https://wordpress.org/plugins/crello/)
Click the &#91;Install Now&#93; button
Click the &#91;Activate&#93; button

== How do I activate the Crello plugin? ==

Contact the Crello team using [https://crello.com/faq/suggest_a_feature/](https://crello.com/faq/suggest_a_feature/), and send us your WordPress domain name.
We will send you a unique API key 
Choose a format you want to use and insert your API key
Click &#91;Save Changes&#93;


Your Crello plugin is ready to use!

== How do I use the Crello plugin? ==

Click on the button &#91;Create in Crello&#93; at the top of the post editing window
Create an image
Click the &#91;Publish&#93; button in the upper right hand corner


For more detailed instructions, read the [documentation section](https://crello.com/crello-button/documentation/).
